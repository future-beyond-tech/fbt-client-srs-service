API Endpoints

RESTful API endpoints for the Sales module. All endpoints require Admin authentication via JWT Bearer token.

POST /api/sales Create new sale transaction
GET /api/sales Get paginated sale history with filters
GET /api/sales/{billNumber} Get detailed sale by bill number
GET /api/sales/{billNumber}/invoice Retrieve invoice data
POST /api/sales/{billNumber}/send-invoice Send invoice via WhatsApp
POST /api/sales/{billNumber}/process-invoice Generate and send invoice

Migration: Twilio ‚Üí Meta WhatsApp

üîÑ Migration Summary

Successfully migrated from Twilio WhatsApp API to Meta WhatsApp Cloud API with zero breaking changes. The IWhatsAppService interface was preserved, ensuring all existing code continues to work without modification.

Aspect Before (Twilio) After (Meta)
Library Twilio NuGet (v7.2.3) Built-in HttpClient + System.Text.Json
Configuration Keys 3 (AccountSid, AuthToken, FromNumber) 2 (AccessToken, PhoneNumberId)
Message Type Direct send with media URL Document message with PDF link
Phone Format whatsapp:+919600433056 E.164 without + (919600433056)
API Version N/A v18.0 (Meta Graph API)
Breaking Changes - ZERO
